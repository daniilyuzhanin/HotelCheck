# Start developing

## Prerequisites

Required Node.js version - 14.10.1

[Install mkcert](https://github.com/FiloSottile/mkcert#installation)

## Installation

Install the project using npm:

```
npm ci
```

## Running

Run the project:

```
npm start
```

### Used port: 3000

Open to browse:

https://localhost:3000

## Build

Build the project:

```
npm run build
```
